# HTools

A collection of my tools for managing and converting * contents

## E-hentai
Set file paths in config.json first. 
### Addinfo.py
Generate ComicInfo.xml to galleries without it. Using gallery names to search on Exhentai to retrieve info. You will have to download your ExHentai cookie.
### GalleryToEhViewer.py
Generate ComicInfo.xml from https://sleazyfork.org/en/scripts/10379-e-hentai-downloader's info.txt and rename the images to %08d format.
